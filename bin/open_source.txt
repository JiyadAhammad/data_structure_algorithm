# Opens Source contribution Guide

-> Choose your language  -> Flutter || Python
-> Find project from github 
-> Check the readMe.md file 
-> Check the contributing.md file
-> initally -> go to issues -> check "GOOD FIRST CONTRIBUTION"
-> fixing error , building new feature , documenting
-> Start the repository
-> fork the repository 
-> create your branch 
-> Write your code in your branch
-> push code into your branch
-> Make a Pull Request to the main branch
-> Respect the feedback of the maintainers
